- max-h-96 overflow-y-auto

+ max-h-[48rem] overflow-y-auto

2. SOPVideoPlayer.tsx - 视频播放器尺寸修改：

- className="w-full rounded-lg mb-4"

+ className="max-w-full max-h-96 rounded-lg mb-4 mx-auto block"

+ style={{ maxWidth: '600px' }}

🎯 用户体验改进

  • 更多内容可见：精修区域高度翻倍，可以同时看到更多区块内容
  • 视频比例合理：纵向视频不会过高，横向视频保持合适尺寸
  • 布局更紧凑：视频播放器居中显示，整体布局更美观
  • 滚动体验更好：更大的滚动区域，减少频繁滚动
